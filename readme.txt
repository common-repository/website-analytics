Wordpress Site Analytics for monitoring wordpress site. Wordpress Site Analytics shows most visited pages, browser, device and platform used.

== Description ==

**Wordpress Site Analytics plugin shows live reports of wordpress site.**

It monitors the complete site and give proper reports.

<h4>FREE Plugin Features</h4>
* Simplified & easy to user interface.
* **User Details:** Logged in status of user with username, logged in time and page visited by user.
* **Visited Page Details:** Give the reports of all visited pages with there referal page.
* **Visited Page count:** Shows the top most visited pages of your site.
* **Comment and Post details:** Keeps report of comments and post with username and time.
* **Device used:** It shows the device used by users i.e. desktop, mobile or tablet.
* **Browser used:** It shows the broswer used by users i.e. Google Chrome, Mozilla Firefox.
* **Platform used:** It shows the platform used by users i.e. Windows, Linux.
* **Error Report:** Give the 404 page not fount error details.

<h4>PREMIUM Plugin Features</h4> 
* **Live Traffic:** It shows currenty active users on your website. with there current visiting page.
* **Total Spent time:** It shows the total time spent on particular web page.
* **Post Details:** It shows top visited post, with author name, post title, total visits on post and total comments on the post.
* **Role based Analytics:** Admin can view the data of selected role of users. for ex:- if subscriber is selected then only those particular users' activities will be shown.
* **Advanced Device details:** It shows all type of devices used by users(role based users). for ex: mobile, desktop etc.
* **Advanced Platform details:** It shows all type of platforms used by users(role based users). for ex: windows, Linux etc.
* **Advanced Browser details:** It shows all type of platforms used by users(role based users). for ex: Google chrome, Mozilla Firefox, Internet Explorer etc.

== Installation ==

= From your Wordpress dashboard =
1. Navigate to `Plugins > Add New` from your WP Admin dashboard.
2. Search for `website analytics`.
3. Install `website analytics` and activate the plugin.

= From Wordpress.org =
1. Search for `website analytics` and download it.
2. Unzip and upload the `website-analytics-for-wp.zip` directory to your `/wp-content/plugins/` directory.
3. Activate website analytics from the Plugins tab of your admin dashboard.
